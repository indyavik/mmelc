Course Assessment Formats (1 of 2)

There are four assessment formats for this course: pre- and post-tests; end of module multiple choice questions; unit assignments; and the final course assessment.

Pre- and Post-Tests

The pre-test and post-test have a similar framework, with distinct questions on the same topics to help you refresh your skills. These tests consist of two questions per unit covering both theoretical and practical aspects of malaria microscopy and use illustrations for slide preparation, stains and RDT results. These tests also contain several virtual microscope slides which we will ask you to examine and record a diagnosis. Please take the pre-test before you commence with the learning activities for Module One so that you will be able to measure your progress.

End-of-Module Multiple Choice Questions

To measure improvements in your knowledge and practical skills, there is a short test comprising 10 multiple choice questions (MCQs) at the end of each module.
