Navigating the Course

As described in the Introduction, the course consists of five modules, each of which has several units. Throughout the Worldwide E-Learning Course on Malaria Microscopy, you will use the methods presented on the following pages to navigate through the content.

To navigate between modules, you can click the links on the first page after logging in or you can use the course contents menu, described below. At the end of each module is a button to continue to the next module.

The course contents menu is opened by clicking on the Menu button in the navigation bar. This opens the menu, with links to each module and unit within the course. This menu also indicates which units and modules you have completed.

The home button provides you with the option of returning to the Course Home Page or the first page of your current unit.

The help button displays a screen containing these navigation tips. When viewing virtual slides, the help button instead displays tips on controlling the Virtual Microscope.

The Logout button logs you out and takes you to the login screen.
