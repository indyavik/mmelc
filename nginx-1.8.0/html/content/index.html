OK Worldwide E-Learning Course on Malaria Microscopy: Getting Started

Participation in the course requires creating a user account, which is done by clicking the"Create New User" button. There are two types of users:

Licensed Users have full access to the course content. Creating a licensed user requires entering a licence key (see below).

Preview Users do not have access to the pre-test, end-of-module tests, nor the post-test.

License Keys must be validated while online. To enter a licence key, click the"Add a new Licence Key" button. If you do not have a licence key but would like to purchase one, please contact Amref International University, http://amref.ac.ke/, or email MalariaMicroscopyCourse@gmail.com for details.

If a licence key is added after creating a preview user, the user can be converted to a licensed user by clicking the"Convert Preview to Licensed User" button.

The final assessment, which must be passed to receive a certificate, requires an additional registration, and can only be taken after completing the entire course as a Licensed User.
