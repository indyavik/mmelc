Module Overview (1 of 2)

Introduction to the Module

Congratulations on completing Module One and welcome to Module Two of the Worldwide E-Learning Course on Malaria Microscopy entitled Blood Collection and Preparation and Staining of Blood Films.

This module is organised around four units and each unit consists of an introduction, learning outcomes, unit content and a demonstration of learning activities using video clips.

At the end of the module you will find a summary of the module and a set of assessment questions to gauge your progress.

Module Purpose

The purpose of this module is to equip you with knowledge and skills to enable you to safely collect blood specimens, prepare quality blood films and correctly stain blood films using recommended quality staining reagents and procedures.
