Preparation of Giemsa Stain Working Solution for Malaria Microscopy (4 of 7)

Preparation of 10% Giemsa Stain Working Solution

The preparation of 10% Giemsa stain working solution is described as follows:

Pipette 9 mL of previously prepared buffered water, pH 7.2, into a clean beaker or tube.

Filter 25–50 mL Giemsa stain stock solution into a clean container, using Whatman #1 filter paper.

Using a clean and dry pipette, add 1 mL of this Giemsa stain stock solution to the buffered water. Do NOT get this stock solution from the big bottle containing the Giemsa stain stock solution to avoid contaminating it.
