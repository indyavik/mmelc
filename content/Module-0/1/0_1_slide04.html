Navigating within the Course Modules

To navigate the slides within each unit, you will mostly use the navigation bar, which contains the following buttons:

The Next button takes you to the next slide within a unit.

The Previous button takes you back one slide within a unit.

To skip to another slide within the unit, you can use the menu at the bottom of the screen.

When you have reached the end of a unit, you will be notified to continue to the next unit by pushing a button. This will mark the unit as completed and send you to the next unit.

The slide navigation tools will not be available during the pre- and post-tests, the post-module-assessments, and the Final Assessment. You must answer each question before continuing to the next, and if you navigate away, log out, or close the software before completion you will need to start again.
